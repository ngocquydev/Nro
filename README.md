# Web Game Nro

Hệ thống hỗ trợ bán nick game (acc) và thanh toán tự động, giúp tối ưu hóa quy trình giao dịch cho người bán và người mua.

## 🚀 Các tính năng chính
* **Thanh toán tự động:** Tự động xác nhận và xử lý giao dịch.
* **Quản lý tài khoản:** Cập nhật thông tin, trạng thái và giá trị nick game.
* **Giao diện hiện đại:** Tối ưu trải nghiệm người dùng với React.

## 🛠 Công nghệ sử dụng

* **Form Handling:** Formik
