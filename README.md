# Web Game Nro

Hệ thống hỗ trợ bán nick game (acc) và thanh toán tự động.

## 🚀 Các tính năng chính
* **Thanh toán tự động:** Tự động xác nhận giao dịch.
* **Quản lý tài khoản:** Cập nhật thông tin và trạng thái nick game.
* *(Đang cập nhật thêm tính năng...)*

## 🛠 Công nghệ sử dụng
* [Node.js](https://nodejs.org/)
* [Express](https://expressjs.com/)
